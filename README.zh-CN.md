## 🍚食用

### 安装

```bash
$ pnpm install
```

### 运行

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### 测试

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

### 构建

要构建服务，请运行

```bash
$ pnpm build
```

您将看到dist中准备提供的生成文件。

### Docker构建运行

```bash
$ docker-compose up --build
```
